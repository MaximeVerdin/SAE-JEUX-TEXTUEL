## How to compile and play

### On Windows

- use the command `gcc -o jeu main.c fonctions/*.c` when in the folder
- run the `jeu.exe` file

### On Linux

- use the command `gcc -o jeu main.c fonctions/*.c` when in the folder
- run the `jeu` file by using the command `./jeu`
